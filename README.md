## Install

### Prerequisites

- [Node.js]() v12.0+
- [Yarn]() v1.22.0+

### Install & development

```sh
# Clone repository
git clone https://github.com/farhan2077/symbols-search

# Change directory
cd symbols-search

# Install dependencies
yarn

# Start the pluto store server
yarn dev
```
